# Dataset

This directory contains the data used for the machine tool health analysis project.

The dataset is not included in this repository due to size and licensing restrictions.

## Source

Dominguez Caballero, Javier Alejandro; Moore, James; Stammers, Jon (2023).

**Sensor signals for machine tool and process health assessment.**

University of Sheffield Dataset Repository.

DOI: https://doi.org/10.15131/shef.data.24125715.v1

## Directory structure

```text
data/
├── raw/
│   └── sensor_signals/
│       └── Original downloaded dataset files
│
└── processed/
    └── Generated datasets after preprocessing
```

## Usage

Download the dataset from the source above and extract the files into:

```text
data/raw/sensor_signals/
```

The analysis scripts will generate processed data outputs automatically.
